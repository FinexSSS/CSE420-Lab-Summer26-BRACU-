# Ultimate Compiler Visual Teaching App

This is a **React and Vite** application designed as the ultimate study guide for the Lexical and Syntax Analyzer lab (Flex/Bison) for CSE420.

It transforms dry code into a fully interactive, baby-level visual experience. 

## Features
- **Theory & Basics**: Plain-English explanations of compilers, lexers, and parsers.
- **100% Code Coverage**: Every single line of the Lexer (`.l`) and Parser (`.y`) files is clickable with detailed explanations.
- **Visual Stack Walkthrough**: A physics-animated, step-by-step visualizer showing exactly how an LALR(1) parser shifts and reduces tokens.
- **30+ Viva Flashcards**: A comprehensive question bank covering theory, memory leaks, operator precedence, and edge cases (like the dangling-else).

## How to Run Locally

To run this on your own machine, you need [Node.js](https://nodejs.org/) installed.

1. Clone the repository and navigate to the project directory:
   ```bash
   cd "Lab 1/visual-teaching-app"
   ```

2. Install the required dependencies:
   ```bash
   npm install
   ```

3. Start the development server:
   ```bash
   npm run dev
   ```

4. Open your browser and go to the link provided in the terminal (usually `http://localhost:5173`).

Enjoy your flawless viva preparation!
